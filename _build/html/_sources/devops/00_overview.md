# DevOps — Overview

This section covers infrastructure, automation, and operations for modern systems.

## Core Tracks
- **Containers & Orchestration** → Docker, Kubernetes, Helm, Kustomize.
- **Cloud** → AWS services, architecture patterns.
- **CI/CD** → GitHub Actions, Jenkins, Argo CD.
- **IaC** → Terraform, Pulumi.
- **Observability** → Prometheus, Grafana, ELK stack.

## How to Use This Section
- Start with **Docker → Kubernetes** if new to orchestration.
- Combine **Cloud + IaC + CI/CD** for end-to-end delivery pipelines.
