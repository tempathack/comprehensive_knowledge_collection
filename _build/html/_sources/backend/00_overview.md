# Backend — Overview

Goal: APIs, services, and system design for robust backends.

## Planned Tracks
- **API Design** → REST, GraphQL, gRPC, versioning, pagination.
- **Communication Patterns** → HTTP request/response, WebSockets, SSE, streaming, sync vs async.
- **Auth & Security** → OAuth2/OIDC, JWT, RBAC/ABAC, secrets.
- **Data Storage & Modeling** → SQL/NoSQL, schema design, indexing, search.
- **Caching & Queues** → cache strategies, pub/sub, background jobs.
- **Resilience & Performance** → timeouts, retries, rate limiting, load balancing.
- **Observability** → logs, traces, metrics, alerting.
- **Testing & Quality** → unit/integration/contract/load testing.
