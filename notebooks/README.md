# Notebooks

This is the core of the repo: explanation-first Jupyter notebooks with plots, intuition, and runnable code.

## Sections
- Data Science: `notebooks/data_science/`
- DevOps: `notebooks/devops/`
- Backend: `notebooks/backend/`
- Cybersecurity: `notebooks/cybersecurity/`
- Frontend: `notebooks/frontend/`

## Notebook Style
- Prefer “teach + show”: short theory blocks followed by visuals and small experiments.
- When relevant, do both:
  - **From scratch** (NumPy) for intuition
  - **In practice** (sklearn/torch/etc.) for real-world workflows
